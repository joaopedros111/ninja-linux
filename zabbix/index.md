# 📊 Zabbix

## Introdução

O Zabbix é uma plataforma de monitoramento de infraestrutura que permite acompanhar servidores, aplicações, serviços, redes e dispositivos em tempo real.

É amplamente utilizado para monitoramento de disponibilidade, desempenho e geração de alertas.
