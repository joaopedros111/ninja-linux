📊 Instalação do Zabbix Server com PostgreSQL no Rocky Linux 9

Este tutorial mostra uma instalação básica do Zabbix Server utilizando PostgreSQL como banco de dados.

Adicionar o repositório do Zabbix

rpm -Uvh https://repo.zabbix.com/zabbix/7.0/rhel/9/x86_64/zabbix-release-latest-7.0.el9.noarch.rpm

dnf clean all

Instalar os pacotes

dnf install -y \
zabbix-server-pgsql \
zabbix-web-pgsql \
zabbix-nginx-conf \
zabbix-sql-scripts \
zabbix-selinux-policy \
zabbix-agent

Instalar PostgreSQL

dnf install -y postgresql-server

Inicializar o banco:

postgresql-setup --initdb

Habilitar o serviço:

systemctl enable --now postgresql

Verificar:

systemctl status postgresql

Criar banco de dados

Acessar o PostgreSQL:

sudo -u postgres psql

Criar usuário:

CREATE USER zabbix WITH PASSWORD 'senha_forte';

Criar banco:

CREATE DATABASE zabbix OWNER zabbix;

Conceder permissões:

GRANT ALL PRIVILEGES ON DATABASE zabbix TO zabbix;

Sair:

\q

Importar o schema do Zabbix

zcat /usr/share/zabbix/sql-scripts/postgresql/server.sql.gz | \
PGPASSWORD='senha_forte' psql -U zabbix -d zabbix

A importação pode levar alguns minutos.

Configurar o Zabbix Server

Editar:

vi /etc/zabbix/zabbix_server.conf

Localizar:

DBPassword=

Alterar:

DBPassword=senha_forte

Salvar o arquivo.

Iniciar serviços

systemctl enable --now zabbix-server
systemctl enable --now zabbix-agent
systemctl enable --now nginx
systemctl enable --now php-fpm

Verificar:

systemctl status zabbix-server

Liberar firewall

firewall-cmd --permanent --add-service=http

firewall-cmd --reload

Acessar interface Web

Abrir no navegador:

http://IP_DO_SERVIDOR

Exemplo:

http://10.0.97.35

Login padrão

Usuário: Admin
Senha: zabbix

Verificações úteis

Verificar logs do servidor:

tail -f /var/log/zabbix/zabbix_server.log

Verificar porta:

ss -tulpn | grep 10051

Verificar banco:

sudo -u postgres psql

Serviços principais

systemctl status zabbix-server
systemctl status zabbix-agent
systemctl status postgresql
systemctl status nginx
systemctl status php-fpm
