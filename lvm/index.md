# 💽 LVM

## Introdução

O LVM (Logical Volume Manager) é uma tecnologia de gerenciamento de armazenamento utilizada em sistemas Linux que permite criar, redimensionar e administrar volumes lógicos de forma flexível.

Com o LVM é possível expandir sistemas de arquivos, adicionar novos discos ao armazenamento existente e gerenciar espaços de forma mais eficiente do que o particionamento tradicional.
